# AutoStream AI Agent

## Overview
This project implements a Conversational AI Agent for AutoStream, a SaaS platform for automated video editing.

The agent can:
- Identify user intent (greeting, question, high intent)
- Answer product questions using RAG
- Detect high-intent users
- Capture leads using a tool

---

## Tech Stack
- Python
- LangGraph
- LangChain
- LM Studio (Mistral 7B)
- FAISS (Vector DB)

---

1. Start LM Studio server  
2. Activate virtual environment
